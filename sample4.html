• This example will set initial height to screen height. However, the blue screen cannot expand when the content overflows even when data-resize attribute is added. To make sure the height is responsive to the content within data-auto attribute must be added.

  • Attributes: data-height="full" data-resize="resize" data-auto="true" or simply data-auto
